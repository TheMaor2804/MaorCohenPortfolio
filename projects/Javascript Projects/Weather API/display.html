Weather Project

This project is a weather app that uses the OpenWeatherMap API, Geocoding API and UVIndex API to get the weather data of any city in the world. The user can search for a city and get the current weather for that city. The app also displays 24 hour forecast and a 5-day forecast for the city. The search input has autocomplete functionality. To display weather for a city it must be clicked on the autocomplete list. (UVIndex API has a limit of 50 requests per day)
